# 🗺️ MoWISE パターン図鑑 — 完全版100パターン
## Pattern Zukan v1.0

> **設計思想**：「捕まえる → 育てる → 使って勝つ」  
> 各パターンはポケモンと同じ構造で管理する。  
> レアリティ★が高いほど、先生（ジムリーダー）のバッジが必要。

---

## 📘 図鑑の見方

| 項目 | 説明 |
|------|------|
| **No.** | 図鑑番号（P001〜P100） |
| **パターン** | 英語の型（`[　]`は入れ替え可能な部分） |
| **日本語訳** | 型の意味 |
| **レアリティ** | ★1〜5（★5が最高レア） |
| **使用シーン** | どの場面で使うか |
| **例文3本** | 実際の使い方サンプル |
| **進化形** | ★5到達で解禁される応用パターン |
| **解禁条件** | 前のパターンが何★以上で出現 |

### AIによる弱点補強の発動条件

| 状態 | AIの動作 |
|------|---------|
| 正解率 < 60% | デイリーに自動追加（本人に通知なし） |
| 反応時間 > 3秒 | スピードドリルを優先出題 |
| 3日間未練習 | ストリーク警告 + 軽いバージョンで再提示 |
| ★5未達のまま次エリア | 先生に「未習得パターン」としてアラート |

- **全体習熟率**：P001〜現在の平均★
- **エリア別進捗**：エリア1〜5の完了率
- **弱点パターントップ3**：正解率の低い順
- **最終練習日**：サボり検知
- **ジムバッジ取得状況**：認定済み/未認定
